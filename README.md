# Data Analyst Portfolio

This repository contains a collection of **Data Analysis projects** focused on data cleaning, transformation, ETL processes, and exploratory data analysis using **Python and SQL**.  
The projects are designed to demonstrate practical, real-world skills relevant to **Data Analyst / BI Analyst** roles.

---

## About This Repository

The main goal of this repository is to showcase my hands-on experience in:

- Data cleaning and preprocessing
- Data transformation and standardization
- ETL workflows
- Exploratory Data Analysis (EDA)
- Structured and reproducible data projects

Each project reflects real scenarios commonly faced in business and analytics environments.

---

## Technologies & Tools

- **Programming:** Python, SQL  
- **Data Processing:** Pandas, NumPy  
- **Databases:** SQL, NoSQL (MongoDB – applied in academic projects)  
- **Data Engineering (Foundations):** ETL pipelines, data modeling  
- **Visualization & BI:** Power BI  
- **Version Control:** Git & GitHub
